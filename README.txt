To run the program :
1. Copy all the files under a directory;
2. Make all the .py file executable, type:
	chmod +x *.py
3. The program will run with the demo data, the full set of data can be downloaded from Kaggle website. The link can be found in the project report
4. Open MySQL server and create a database, use the sql script in TabelCreation.sql to create 2 tabel schema
5. to run the program, type in bash:
	./PipelineMain.py
5. all the analysis and result will be generated by the program.
